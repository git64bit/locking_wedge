# Changelog

- Established the Locking Wedge project from base commit `b054679`.
- Added a parametric 200 mm wedge with configurable width.
- Established 7 mm and 13 mm end thicknesses.

- Retained transverse female dovetail mounting choices.
- Corrected the female coupons to use the production two-ended entry relief.
- Removed the printed hex locator.
- Added distinct lower-base and upper-slide models.
